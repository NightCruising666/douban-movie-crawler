# 数据字典

## movies_raw.csv

| 字段 | 含义 |
|---|---|
| 豆瓣ID | 电影稳定主键 |
| 电影名称 | 搜索接口返回名称 |
| 搜索评分 | 搜索阶段的评分快照 |
| URL | 豆瓣电影链接 |
| 采集时间 | 带时区的 ISO 8601 时间；历史数据可为空 |

## movie_tags.csv

| 字段 | 含义 |
|---|---|
| 豆瓣ID | 电影主键 |
| 标签 | 搜索标签 |
| 标签内排名 | 采集时该标签内的返回顺序 |
| 采集时间 | 标签快照时间 |

## movies.csv

| 字段 | 含义 |
|---|---|
| 豆瓣ID | 电影主键 |
| 电影名称 / 原始片名 | 中文名与 API 原始片名 |
| 导演 / 主演 | 多值用 ` / ` 分隔，主演最多15人 |
| 上映年份 / 首映日期 | 时间字段 |
| 类型 / 国家/地区 | 多值分类字段 |
| 片长 | API 原始片长文本 |
| 豆瓣评分 | 0—10 分快照 |
| 评价人数 | 打分用户总数快照 |
| 短评总数 / 长评总数 | API 统计快照 |
| 采集时间 | 详情快照时间 |

## reviews.csv

| 字段 | 含义 |
|---|---|
| 短评ID | 短评稳定主键 |
| 豆瓣ID / 电影名称 | 关联电影 |
| 评分 | 1—5星或未评分 |
| 短评正文 | 公开短评文本 |
| 有用数 | 采集时的社区有用投票数 |
| 评论时间 | 短评发布时间 |
| 采样方式 | 当前固定为热门 |
| 排序位置 | 该采样口径中的顺序 |
| 采集时间 | 短评快照时间 |

## detail_failure_attempts.csv

| 字段 | 含义 |
|---|---|
| 豆瓣ID / 电影名称 | 失败电影主键与名称 |
| 轮次 / 轮内尝试序号 | 区分补采轮次和长冷却后的重试 |
| 失败原因 | HTTP 状态或网络/解析错误 |
| 失败时间 | 该次电影级请求失败的审计时间 |

## detail_failures.csv / unavailable_movies.csv

| 字段 | 含义 |
|---|---|
| 豆瓣ID / 电影名称 | 失败电影主键与名称 |
| 最后失败原因 | HTTP 状态或网络/解析错误 |
| 首次失败轮次 / 最后失败轮次 | 用于区分不同补采轮次 |
| 失败轮次数 | 不同轮次的失败总数，同轮重复不重复计数 |
| 永久失败轮次数 | HTTP 400/404/410 的失败轮数 |
| 连续永久失败轮次数 | 连续永久失败达到 2 才判定不可用 |
| 状态 | 待复核、已恢复或不可用 |
| 首次失败时间 / 最后更新时间 | 带时区的采集审计时间 |

`unavailable_movies.csv` 是 `detail_failures.csv` 中状态为“不可用”的子集。网络错误、HTTP 403/429 和 JSON 解析错误不会被直接判定为永久不可用。

## 派生指标

派生指标只写入 `data/processed/`，不修改原始采集文件。

- 短评参与率 = 短评总数 / 评价人数
- 长评参与率 = 长评总数 / 评价人数
- 贝叶斯加权评分 = `v/(v+m)*R + m/(v+m)*C`
- 对数有用数加权星级：权重 `1 + log(1+有用数)`
- 样本五星占比及 Wilson 95% 置信区间
- 星级分布熵

新版数据不采集用户名、用户 ID、头像地址及其哈希。旧版归档短评表也已删除用户标识列。
