# 统计分析方案

## A. 评分统计稳定性

1. **评分分布**：直方图、KDE、均值、中位数、分位数。
2. **样本量关系**：`log10(评价人数+1)` 与评分的散点、Spearman 相关和分组方差。
3. **贝叶斯收缩**：比较原评分与贝叶斯加权评分，识别样本量小的极端高分。
4. **四象限**：评分阈值与评价人数分位数组合，并对阈值做敏感性分析。
5. **参与率**：短评/长评参与率与评分、评价人数的关系。
6. **短评一致性**：对每部电影的热门短评样本计算普通平均、原始有用数加权、对数加权、分布熵，并与电影总体评分比较。
7. **不确定性**：样本五星占比必须同时报告 95% 置信区间。

不能仅凭评价人数少、五星多或加权差异直接断言刷分。

## B. 豆瓣样本的类型结构变化

1. 限定 2005—2025 年，先报告每年样本数；样本过少时改用三年滑动窗口或五年分组。
2. 多类型电影按 `1/类型数` 分配权重，避免一部电影对总数重复贡献。
3. 计算年度加权类型占比、类型评分中位数和评价人数中位数。
4. 产地分为中国大陆、合拍、港澳台、进口和未知，避免将合拍片强行二分。
5. 对评分、评价人数的产地差异同时报告效应量，不只报告箱线图。

## 样本代表性检查

- 每年电影数、标签来源数和标签内排名分布。
- 各年份在“热门/最新/经典/高分/地区/类型”标签中的覆盖差异。
- 报告热门前 30 条的排序偏差；接口参数验证发现所谓时间参数与热门结果高度重合，因此不作为独立样本层。
- 缺失值、重复值和 API 穷尽电影的比例。

## 计划图表

1. 评分直方图 + KDE
2. `log10(评价人数+1)` — 评分散点与分组方差
3. 原评分—贝叶斯加权评分差值图
4. 高分/低分 × 大众/小众四象限
5. 短评三种评分口径对比
6. 2005—2025 加权类型占比面积图
7. 产地分类的评分与评价人数箱线图
8. 类型—产地热力图

## 输出与解释边界

- `data/analysis/analysis_summary.csv` 保存核心统计量与效应量。
- `movie_quadrants.csv` 保存评分高低、受众大众/小众的样本内四象限分类。
- `quadrant_sensitivity.csv` 使用多个评分阈值和评价人数分位数检验结论是否依赖单一划分。
- `yearly_genre_share.csv` 对多类型电影采用 `1/类型数` 权重，单部电影总权重为 1。
- Spearman 相关描述单调关系，不代表因果；Cohen's d 只描述样本组间标准化差异。
- 评价人数、短评总数、长评总数是采集时点快照，随时间会变化。
- 数据质量部分报告阶段二确认不可用数量和比例；不可用条目不进入评分与类型统计，但失败证据随项目归档。
